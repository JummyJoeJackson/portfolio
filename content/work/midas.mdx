# Midas: AI Fraud Detection

Midas is a high-end fraud detection prototype designed to help financial institutions identify and prevent fraudulent transactions in real-time.

## Features

- Real-time transaction monitoring
- Predictive analysis using YOLO-based models
- Premium glassmorphism UI for data visualization

Built with FastAPI and Next.js.
